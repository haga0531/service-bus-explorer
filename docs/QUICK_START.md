### Message Operations
- **Peek**: View messages without consuming
- **Delete**: Remove messages
- **Resubmit**: Resend dead letters
- **Purge**: Delete all messages

### Shortcuts
- `Cmd+K`: New connection
- `Cmd+R`: Refresh
- `Delete`: Delete selected messages
